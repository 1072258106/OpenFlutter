# OpenFlutter

## What's OpenFlutter

OpenFlutter is a open source organization of Flutter.OpenFutter is a central place for all community made Flutter packages.

---

## 什么是 OpenFlutter

OpenFlutter 是一个 Flutter 的开源组织。OpenFutter 致力于成为第三方 Flutter 包集中地。
